# PSK Travel Agency

## Project Documentation

### Overview
PSK Travel Agency is a web application designed to facilitate travel bookings and management for users. The application allows users to search for flights, hotels, and car rentals, making it an all-in-one solution for travel planning.

### Features
- **User Registration & Login**: Secure user authentication to manage bookings.
- **Search Functionality**: Search for flights, hotels, and car rentals based on various criteria.
- **Booking Management**: View and manage existing bookings.
- **Payment Integration**: Secure payment processing for bookings.

### Technologies Used
- **Frontend**: HTML, CSS, JavaScript, React
- **Backend**: Node.js, Express, MongoDB
- **Payment Gateway**: Stripe API

### Installation Guide
1. Clone the repository: `git clone https://github.com/ash142003/psk-travel-agency.git`
2. Navigate to the project directory: `cd psk-travel-agency`
3. Install dependencies: `npm install`
4. Start the app: `npm start`
